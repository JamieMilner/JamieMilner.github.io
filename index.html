Personal Projects & Achievements

Automated Dynamics CRM Integration

In my current role, I have built a system from scratch that can create, update and delete entities on CRM based on changes detected in EF Core automatically. This (in conjunction with webhooks) allows for client CRMs to be kept in sync with the website/portal database at all times while minimizing traffic between the systems from extraneous data being sent. This works through using generic methods and custom built models and properties that are mapped to domain models and then used to generate the appropriate request to be transmitted.

Pix 'N Stix

This is Mobile application that I developed for my university project dissertation. It is an app that is designed as a bespoke social media platform for musicians and bands to allow them to connect easily, form bands, and find shows to play. The app is built using traditional front-end languages, but utilizes the Cordova Framework to function on Android devices. As well as the traditional front-end, this app also uses multiple PHP scripts located on a XAMPP local server, to interact with a MySQL database, which gives my app persistent storage.
